📝 기획 의도
-------------
<p>현재 젊은 세대에서는 한정판과 콜라보, 발매가 중단된 상품이 인기를 끌고 있습니다.</p>
<p>크림은 익명거래를 통하여 안전한 거래를 보장하고, 시세를 예측할 수 있는 시스템으로 현명한 거래를 제공합니다.</p>
<p>그래서 리셀 거래 중개 플랫폼 크림은 구매자에게 희소성 있는 상품이나 원하는 상품의 구매 방도를 마련합니다.</p>
